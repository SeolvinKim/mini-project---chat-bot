# PRD_TEMPLATE.md - 금융권 직무 추천 툴

## 1. 기능 요구사항 (Functional Requirements)

- Gradio 입력창에 자연어 프로필을 입력하면 응답이 출력된다.
- 금융권 직무 7개 후보 중 상위 3개를 추천한다.
- 추천 이유, 필요한 역량, 준비 방향이 모두 포함된다.
- 부족한 입력에 대해 추가 질문을 반환한다.

---

## 1.1 입력값

사용자는 챗봇에게 다음 정보를 자연어로 입력할 수 있다.

- 전공
- 관심 금융 분야
- 보유 경험
- 보유 역량
- 보유 자격증
- 선호하는 업무 방식

예시:

```text
저는 금융권에 관심이 있고 데이터 분석을 배우고 있습니다. SQL과 Python을 공부 중이며, 고객 데이터를 활용한 분석 업무를 해보고 싶습니다.
```

```text
경제학과이고 주식 투자와 기업 분석에 관심이 많습니다. 리서치나 자산관리 쪽 직무가 궁금합니다.
```

---

## 1.2 출력값

툴은 사용자 입력을 바탕으로 다음 정보를 제공한다.

- 추천 직무 3개
- 각 직무별 추천 이유
- 해당 직무에서 필요한 역량
- 앞으로의 준비 방향

기본 응답 형식:

```text
현재 입력 기준으로는 다음 3개 직무가 잘 맞을 수 있습니다.

1. 직무명
- 추천 이유:
- 필요한 역량:
- 준비 방향:

2. 직무명
- 추천 이유:
- 필요한 역량:
- 준비 방향:

3. 직무명
- 추천 이유:
- 필요한 역량:
- 준비 방향:

마무리 조언:
```

---

## 1.3 추천 직무 범위

MVP에서 추천하는 금융권 직무는 다음 7개로 제한한다.

| ID | 추천 직무 | 주요 설명 |
|---|---|---|
| `finance_data_analyst` | 금융 데이터 분석가 | 고객, 거래, 상품, 리스크 데이터를 분석해 의사결정을 지원 |
| `retail_sales` | 리테일 금융/증권 영업 | 개인 고객 대상 금융상품 상담, 계좌/상품 안내, 영업 관리 |
| `pb_wm` | PB/WM 자산관리 | 고객의 자산 현황과 투자 성향을 바탕으로 포트폴리오 상담 |
| `financial_platform_pm` | 금융 플랫폼 기획자 | 금융 앱/웹 서비스의 기능, UX, 사용자 흐름을 기획 |
| `risk_credit` | 리스크 관리/여신 심사 | 신용도, 대출 가능성, 연체 가능성, 금융 리스크를 검토 |
| `research_investment` | 리서치/투자분석 | 경제, 산업, 기업 데이터를 분석해 투자 판단 자료 작성 |
| `compliance_internal_control` | 컴플라이언스/내부통제 | 금융 관련 법규, 내부 규정, 사고 예방 체계 점검 |

## 2. 추천 로직

### 2.1 1차 추천 방식

사용자의 입력 문장에 포함된 키워드를 기준으로 직무별 적합도를 계산한다.

| 사용자 입력 키워드 | 우선 추천 후보 |
|---|---|
| 데이터, 분석, SQL, Python, 통계, 시각화 | 금융 데이터 분석가, 리스크 관리/여신 심사, 리서치/투자분석 |
| 고객, 상담, 소통, 영업, 대면 | 리테일 금융/증권 영업, PB/WM 자산관리 |
| 투자, 주식, 경제, 시장, 포트폴리오 | 리서치/투자분석, PB/WM 자산관리 |
| 플랫폼, 앱, 디지털, 핀테크, 서비스, UX | 금융 플랫폼 기획자 |
| 꼼꼼함, 리스크, 신용, 심사, 대출 | 리스크 관리/여신 심사 |
| 규정, 준법, 내부통제, 법, 감사 | 컴플라이언스/내부통제 |

### 2.2 점수 계산 규칙

1. 사용자 입력을 소문자 기준으로 정규화한다.
2. 직무별 키워드가 입력에 포함되면 점수를 더한다.
3. 전공, 경험, 역량, 선호 업무 방식이 직무 설명과 맞으면 가중치를 더한다.
4. 상위 3개 직무를 추천한다.
5. 동점이면 다음 우선순위를 적용한다.
   - 사용자가 명시한 관심 분야와 더 가까운 직무
   - 사용자가 보유한 기술이나 경험과 직접 연결되는 직무
   - 취업준비생이 준비 방향을 세우기 쉬운 직무

### 2.3 부족한 입력 판단

다음 조건 중 2개 이상에 해당하면 추천 대신 추가 질문을 반환한다.

- 입력 길이가 20자 미만
- 전공, 관심 분야, 경험, 역량 중 어떤 정보도 없음
- 금융권 직무와 관련된 키워드가 없음
- 사용자의 질문이 직무 추천과 무관함
